Requirements

  • Integrated and Interactive Mapping System: User must be able to view a map of Africa with estimated forest elephant poaching events.
  • Built-in Carbon Calculator Tool: Allows the user to calculate the carbon footprint of a flight and suggest a donation amount.
  • Suggested Donation Links: Provides links to pre-existing elephant conservation organizations.
  • Administrative Login from the Web: Administrative login to change and update the data from the MIKE database.
  • Backend Database Management: Provides up-to-date (within a year) information for our systems to run on.
  • Data from the MIKE Database: Uses data from the MIKE database by uploading a CSV file and loading it into the application database.
  • Data Accuracy: Ensure the application is supplying the user with current information from the annually updated MIKE Database.
  • Timing Characteristics: Ensure the user gets a reasonable response time.
  • Cross Platform Compatibility: Working on Android and iOS platforms, including iOS iPad as a stretch goal.
